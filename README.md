## 🗂️ Overview  

This repository is a **learning/practice** collection of small Java programs used to understand and experiment with core Java concepts.

---

## 📘 Topics Covered

| No. | Topic                           | Description |
|-----|---------------------------------|-------------|
| 01  | 📗 Java Basic Programs           | Fundamental Java programs (syntax, loops, conditionals, etc.) |
| 02  | 🏷️ OOPs - Class & Object         | Programs related to classes, objects, methods, and constructors |
| 03  | 🏛️ OOPs - Inheritance            | Demonstrating single, multilevel, and hierarchical inheritance |
| 04  | 🎭 OOPs - Polymorphism           | Compile-time (method overloading) and runtime (method overriding) polymorphism |
| 05  | 🔒 OOPs - Encapsulation          | Data hiding and getter/setter implementations |
| 06  | 🧩 OOPs - Abstraction            | Abstract classes and interfaces |
| 07  | 🔄 `this` Keyword                | Usage of `this` keyword in constructors and methods |
| 08  | ⬆️ `super` Keyword               | Using `super` to call parent constructors and methods |
| 09  | ⚖️ super vs this                 | Difference between `super` and `this` |
| 10  | 🔀 Type Casting                  | Widening and narrowing type casting |
| 11  | 🛡️ Access Modifiers              | Public, private, protected, and default access levels |
| 12  | 📌 Static Programs               | Static variables, methods, and blocks |
| 13  | 🧾 Enum                          | Enum usage in Java |
| 14  | 🎁 Wrapper Class                 | Autoboxing and unboxing |
| 15  | ⚠️ Exceptions                    | Exception handling using try-catch, finally, and throw/throws |
| 16  | 🧮 Generics                      | Generic classes, methods, and collections |
| 17  | 📚 Collection Framework          | List, Set, Map, and other collection interfaces and classes |
| 18  | 🧵 Multi-Threading               | Creating and managing threads, synchronisation, and concurrency |

---

## ⚙️ Tech Stack

- **Java 11+ (JDK)** — primary language and runtime
- **IntelliJ IDEA** — IDE for writing, running, and debugging Java programs
- **JVM** — to execute compiled `.class` files
- **Enums & Wrapper Classes** — additional core features of Java
- **Exception Handling Mechanisms** — try-catch-finally, throw/throws
- **Generics** — type-safe data structures and methods
- **Java Collections Framework (JCF)** — to practice lists, sets, maps, and queues
- **Multithreading & Concurrency APIs** — to explore parallel execution and synchronization
